## 🎯 方法二：手动命令行操作

### 步骤：

1. **打开命令行**：
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**：
   ```cmd
   cd /d E:\phpstudy_pro\WWW\life_recorder
   ```

3. **添加文件**：
   ```cmd
   git add .
   ```

4. **提交代码**：
   ```cmd
   git commit -m "修复每日提醒、隐私锁和生物识别功能"
   ```

5. **推送到 GitHub**：
   ```cmd
   git push -u origin main
   ```

6. **输入 GitHub 账号密码**（如果需要）：
   - 用户名：你的 GitHub 用户名
   - 密码：你的 GitHub 密码或个人访问令牌

---

## 🎯 方法三：使用 GitHub Desktop（最简单）

### 步骤：

1. **打开 GitHub Desktop**

2. **添加本地仓库**：
   - File → Add Local Repository
   - 选择文件夹：`E:\phpstudy_pro\WWW\life_recorder`
   - 点击"Add Repository"

3. **查看更改**：
   - 左侧会显示所有修改的文件
   - 确认文件都已选中

4. **填写提交信息**：
   - Summary（摘要）：`修复每日提醒、隐私锁和生物识别功能`
   - Description（描述）：`优化通知权限、PIN 码验证和生物识别逻辑`

5. **点击提交**：
   - 点击"Commit to main"按钮

6. **推送代码**：
   - 点击右上角"Push origin"按钮
   - 等待上传完成

## ✅ 推送成功后

### 1. 查看构建进度

访问 GitHub Actions 页面：
```
https://github.com/jiege455/life_recorder/actions
```

### 2. 等待构建完成

- 构建通常需要 **5-15 分钟**
- 刷新页面查看进度
- 绿色 ✓ 表示成功
- 红色 ✗ 表示失败

### 3. 下载 APK

1. 点击成功的构建任务
2. 滚动到页面底部
3. 在"Artifacts"部分点击 `release-apk`
4. 下载完成后解压
5. 得到 `app-release.apk`

### 4. 安装测试

1. 将 APK 传输到手机
2. 在手机上安装
3. 测试所有修复的功能：
   - ✅ 每日提醒（点击测试通知）
   - ✅ 隐私锁（PIN 码验证）
   - ✅ 生物识别（指纹/面部）

## 🔧 常见问题

### Q1: 推送时提示认证失败
**解决方法**：
- 使用 GitHub 个人访问令牌代替密码
- 或者在 GitHub Desktop 中重新登录

### Q2: 推送时提示权限不足
**解决方法**：
- 确认你是仓库所有者
- 或者你有写入权限

### Q3: Git 命令未找到
**解决方法**：
- 安装 Git：https://git-scm.com/download/win
- 安装后重启命令行

### Q4: 推送速度慢
**解决方法**：
- 检查网络连接
- 可能需要使用加速器
- 或者稍后重试
